# cyber-fish-tank 🐟

픽셀 아트로 만든 가벼운 관상용 사이버 어항입니다.

레트로 브라운관 느낌의 유리 어항 속에서 물고기들이 유유히 헤엄칩니다.
흔들리는 수초와 산호, 올라오는 기포, 은은한 스캔라인까지 —
켜두고 바라보기만 해도 좋지만, 자세히 보면 꽤 많은 일이 벌어지고 있습니다.

**Live**: https://wonbang.github.io/cyber-fish-tank/

## 어항 속 생물들

- 🐠 물고기 13종 — 테트라, 구피, 흰동가리, 엔젤피시, 복어, 해파리, 게, 황새치, 고래, 해마, 불가사리, 희귀한 황금 물고기, 그리고 최고급 벨루가
- 🎨 12가지 색상 조합, 태어날 때마다 다른 모습
- 🌿 켈프 · 잔디 · 덤불 · 산호 · 빛나는 광섬유 식물이 자라는 바닥 정원
- 🌙 저녁이 되면 어항도 함께 밤을 맞습니다 (달빛, 발광 플랑크톤)

## 할 수 있는 것

- 🍞 **먹이 주기** — 물을 클릭하면 먹이가 떨어집니다. 서로 먼저 먹으려 다투고, 바닥에 떨어진 부스러기는 게가 치웁니다
- 💗 **이름 지어주기** — 먹이를 세 번 챙겨준 물고기는 하트를 띄웁니다. 클릭해서 이름을 지어주세요
- 💰 **보물상자** — 클릭하면 금화가 쏟아집니다. 단, 상자를 지키는 맨티스 쉬림프에게 너무 가까이 간 물고기는 펀치에 맞아 날아갑니다
- 🥚 **번식** — 사이좋은 두 마리가 알을 낳고, 알에서 새 생명이 태어납니다
- 🚔 **감옥** — 물고기를 드래그해서 왼쪽 아래 감옥에 넣으면, 형기를 마칠 때까지 런닝머신 위를 달려야 합니다

## 골드와 상점 🪙

어항에 떨어지는 모든 금화는 이제 지갑에 쌓입니다. 보물상자를 열고, 레이드에서 이기고,
도감에 새 생물을 등록할 때마다 골드가 들어옵니다. 왼쪽 패널의 상점에서 쓸 수 있습니다:

- 🍞 **먹이** — 일반 · 고급 · 황금 사료. 물 클릭 한 번에 재고가 몇 알씩 소모됩니다. 배부른 물고기만 번식할 수 있고, 비싼 사료를 먹일수록 좋은 등급의 알을 낳을 확률이 올라갑니다
- 🥚 **물고기 알** — 일반 · 고급 · 전설 등급의 랜덤 가챠 알. 등급이 높을수록 희귀종 확률이 오르고(확률 공개), 어떤 종이 나올지는 부화하는 순간 정해집니다
- 🪸 **장식물** — 모아이 석상, 네온 사인, 성, 난파선. 설치 후 드래그로 자리를 옮길 수 있습니다
- 🖼️ **어항 테두리** — 원목 · 네온 · 황금 프레임. 한 번 잠금 해제하면 언제든 갈아끼울 수 있습니다
- 🏷️ **판매 모드** — 어항이 가득 차면(최대 30마리) 물고기를 클릭해 되팔 수 있습니다. 황금 물고기는 비싸게 쳐줍니다

## 도감 📖

어항에 들어온 생물은 종별로 도감에 기록됩니다. 아직 못 만난 종은 실루엣으로만 보입니다.
테트라 · 구피 · 엔젤피시 · 해마는 12가지 색상 변형이 있어서, 전부 모으는 게 진짜 도전입니다.
신종 발견과 새 색상 발견마다 보너스 골드가 들어옵니다.

진행 상황(골드 · 도감 · 장식물 · 테두리)은 브라우저에 저장되어 다시 방문해도 그대로입니다.

## 상어 레이드 🦈

가끔 물이 어두워지고 멀리서 지느러미 그림자가 지나가면 — 거대 상어가 옵니다.

어항 전체가 맞서 싸웁니다. 복어는 가시 방패로 버티고, 맨티스 쉬림프는 포탑이 되고,
물고기 떼는 파도처럼 들이받습니다. 클릭으로 지휘할 수도 있습니다:

- 상어 근처 클릭 → 일제 돌격
- 보물상자 클릭 → 맨티스 조기 출격
- 먼 곳 클릭 → 산개 (물어뜯기 회피)

이기면 금화 비와 함께 희귀한 알이 떨어집니다. 버티지 못하면… 한 마리를 잃습니다.
