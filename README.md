# cyber-fish-tank 🐟

픽셀 아트로 만든 가벼운 관상용 사이버 어항입니다.

레트로 터미널 창 안에서 물고기들이 유유히 헤엄칩니다.
흔들리는 수초와 산호, 올라오는 기포, 은은한 CRT 스캔라인까지 —
켜두고 바라보기만 해도 좋은 작은 어항입니다.

**Live**: https://wonbang.github.io/cyber-fish-tank/

## 즐길 거리

- 🐠 8종의 물고기, 12가지 색상 조합
- 🌿 켈프 · 잔디 · 덤불 · 산호 · 광섬유 식물이 자라는 바닥 정원
- 🍞 물을 클릭하면 먹이가 떨어지고 물고기들이 모여듭니다
- 💰 모래 위 보물상자를 클릭하면 금화가 쏟아집니다
- 🦈 가끔 상어가 나타나 어항이 술렁입니다
- 🌙 야간 모드 — 달빛과 발광 플랑크톤, 네온 수초

의존성 없는 HTML 파일 하나. 빌드 없이 브라우저에서 바로 열립니다.
