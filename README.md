# 픽셀어항 · Pixel Aquarium 🐟

픽셀 아트로 만든 가벼운 관상용 사이버 어항입니다.

레트로 브라운관 느낌의 유리 어항 속에서 물고기들이 유유히 헤엄칩니다.
흔들리는 수초와 산호, 올라오는 기포, 은은한 스캔라인까지 —
켜두고 바라보기만 해도 좋지만, 자세히 보면 꽤 많은 일이 벌어지고 있습니다.

**Live**: https://wonbang.github.io/cyber-fish-tank/

## 다국어 지원 🌐

한국어와 영어를 지원합니다. 브라우저(또는 macOS 시스템) 언어를 자동 감지해 시작하며,
macOS 앱에서는 메뉴바 🐟 아이콘 우클릭 → **언어(Language)** 메뉴에서 한국어/English를 직접 선택할 수 있습니다.
선택한 언어는 저장되어 다음 실행에도 유지됩니다.

Korean and English are supported. The language is auto-detected from your browser
(or macOS system) locale; in the menubar app, right-click the 🐟 icon → **Language** to switch.

## 어항 속 생물들

- 🐠 **물고기 50종** — 테트라·구피부터 아로와나·실러캔스·개복치 같은 전설, 흰수염고래·대왕오징어·벨루가 같은 신화급까지. 심해층을 열어야 만나는 심해 어종과, 그 계절에만 알에서 나오는 계절 한정 어종(봄 산천어·참돔·나비고기, 여름 고등어·날치·만타가오리, 가을 연어·전어·갈치, 겨울 대구·빙어·방어)도 있습니다
- 🎨 테트라·구피·엔젤피시·해마는 12가지 색상 변형 — 총 48가지 색을 도감에 모을 수 있습니다
- 🌿 켈프 · 잔디 · 덤불 · 산호 · 빛나는 광섬유 식물이 자라는 바닥 정원
- 🌙 저녁이 되면 어항도 함께 밤을 맞습니다 (달빛, 발광 플랑크톤)

## 할 수 있는 것

- 🍞 **먹이 주기** — 먹이 버튼으로 사료를 고르고 물을 클릭하면 떨어집니다. 서로 먼저 먹으려 다투고, 바닥에 떨어진 부스러기는 게가 치웁니다
- 💗 **이름 지어주기** — 먹이를 세 번 챙겨준 물고기는 하트를 띄웁니다. 클릭해서 이름을 지어주세요
- 👑 **왕관** — 먹이를 열 번 먹은 고참은 금색 왕관을 씁니다. 판매가 +50%, 번식 시 상위 등급 알 확률이 올라갑니다 (부모 한 마리당 +10%p, 사료 등급 상한은 그대로). 오래 굶기면 왕관이 빛을 잃으니 계속 챙겨주세요
- 💰 **보물상자** — 클릭하면 금화가 쏟아집니다. 단, 상자를 지키는 맨티스 쉬림프에게 너무 가까이 간 물고기는 펀치에 맞아 날아갑니다
- 🥚 **번식** — 배부른 두 마리가 알을 낳고, 알에서 새 생명이 태어납니다
- 🚔 **감옥** — 물고기를 드래그해서 왼쪽 아래 감옥에 넣으면, 형기를 마칠 때까지 런닝머신 위를 달려야 합니다. 노역을 마치면 20~40골드를 벌어다 줍니다
- 😴 **방치 보상** — 앱을 꺼둔 시간도 계산됩니다. 다시 열면 그동안의 골드와 부화가 반영됩니다

## 골드와 상점 🪙

어항에 떨어지는 모든 금화는 지갑에 쌓입니다. 보물상자를 열고, 레이드에서 이기고,
도감에 새 생물을 등록할 때마다 골드가 들어옵니다. 상점에서 쓸 수 있습니다:

- 🍞 **먹이** — 일반·고급 사료는 매일 00시에 열리는 무료 배급(하루 20회/10회)이 끝나면 골드로 구매할 수 있고, 황금 사료는 항상 유료입니다. 물 클릭 한 번에 재고가 몇 알씩 소모됩니다. 배부른 물고기만 번식할 수 있고, 좋은 사료를 먹일수록 좋은 등급의 알을 낳을 확률이 올라갑니다
- 🥚 **물고기 알** — 일반(100) · 고급(300) · 전설(900) · 신화(2500) 등급의 랜덤 가챠 알. 등급이 높을수록 희귀종 확률이 오르고(확률 공개), 어떤 종이 나올지는 부화하는 순간 정해집니다. 신화 알은 전설급 이상만 나옵니다
- 🖼️ **어항 테두리** — 원목(300) · 황금(800) · 네온(1500) 프레임. 한 번 잠금 해제하면 언제든 갈아끼울 수 있습니다
- 🌊 **어항 확장** — 심해층을 개방하면 어항이 아래로 깊어집니다 (층당 +8마리, 최대 2층 = 46마리). 심해는 어둡고 발광 플랑크톤이 반짝이며, 층을 열 때마다 그곳에만 사는 심해 어종이 알에서 나오기 시작합니다
- 🏷️ **판매** — 물고기를 클릭해 정보 카드에서 되팔 수 있습니다. 왕관 쓴 물고기는 +50% 비싸게 쳐줍니다 (최소 7마리는 남겨야 합니다)

## 도감과 업적 📖🏆

어항에 들어온 생물은 종별로 도감에 기록됩니다. 아직 못 만난 종은 실루엣으로만 보이고,
계절 한정 어종은 해당 계절이 표시됩니다. 신종을 발견하면 축하 카드와 함께 보너스 골드가 들어옵니다.

업적 탭에서는 먹이 주기 · 알 부화 · 판매 · 레이드 · 노역 · 보물상자 · 도감 · 계절 한정 · 색상 수집
9종의 업적을 단계별로 달성하고 골드 보상을 받을 수 있습니다.
기록 탭에는 이벤트 로그와 최근 부화 내역이 남습니다.

진행 상황은 브라우저에 저장되어 다시 방문해도 그대로입니다.

## 상어 레이드 🦈

가끔 물이 어두워지고 멀리서 지느러미 그림자가 지나가면 — 거대 상어가 옵니다.

어항 전체가 맞서 싸웁니다. 복어는 가시 방패로 버티고, 맨티스 쉬림프는 포탑이 되고,
물고기 떼는 파도처럼 들이받습니다. 클릭으로 지휘할 수도 있습니다:

- 상어 근처 클릭 → 일제 돌격
- 보물상자 클릭 → 맨티스 조기 출격
- 먼 곳 클릭 → 산개 (물어뜯기 회피)

이기면 금화 비와 함께 희귀한 알이 떨어집니다. 버티지 못하면… 한 마리를 잃습니다.

## 실행 / 개발

Vite + TypeScript 프로젝트입니다. 빌드는 단일 HTML 파일로 나옵니다.

```
npm install
npm run dev      # 개발 서버 (HMR)
npm run build    # dist/ 정적 빌드 — 브라우저에서 dist/index.html 바로 열림
```

## macOS 메뉴바 위젯 🖥️

메뉴바의 🐟 아이콘을 누르면 팝오버로 어항이 열립니다. 우클릭 메뉴에서 창으로 분리하거나 언어를 바꿀 수 있습니다.

**설치 (Releases)**

1. [Releases](https://github.com/WonBang/cyber-fish-tank/releases)에서 `CyberFishTank-vX.Y.Z.zip` 다운로드 후 압축 해제
2. `CyberFishTank.app`을 **우클릭 → 열기** (아직 노터라이즈 전이라 첫 실행은 이 방법이 필요합니다. 이후엔 그냥 실행됩니다)

**직접 빌드**

```
./macos/build.sh run
```

Xcode Command Line Tools만 있으면 됩니다. `./macos/build.sh release`는 배포용 zip을 만듭니다.
